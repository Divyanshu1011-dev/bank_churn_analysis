# 🏦 Bank Customer Churn Analysis Dashboard

## Overview
This project analyzes customer churn behavior in a banking environment using Power BI. The dashboard helps identify factors influencing customer attrition and supports data-driven retention strategies.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key KPIs
- Total Customers
- Churn Rate
- Active Customer %
- Average Credit Score
- Average Balance
- Average Tenure

## Dashboard Features
### Executive Overview
- Customer distribution
- Churn overview
- Key KPIs

### Customer Analysis
- Churn by Geography
- Churn by Gender
- Churn by Age Group

### Retention Analysis
- Active vs Inactive Customers
- Product Holding Analysis
- Balance Analysis

## Key Insights
- Middle-aged customers showed higher churn rates.
- Inactive customers were more likely to leave.
- Geography had a significant impact on churn behavior.
